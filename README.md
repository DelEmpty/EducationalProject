Rules
- If you want to join the repo, you must create your own branch based on the branch master
- Pull requests are mandatory
- Javadoc is mandatory



Project requirements:
- Java JDK 21
- IntellijIDEA Community/Ultimate is reccomended, but you can also use Visual Studio Code




Commands useful you must know

mvn clean install (reinstall all the dependencies from the pom.xml)
mvn build compile (" ")
git add . (Add all the modifies to a single commit)
git commit -m "<message to commit>" (a message that describes the actions you made
git push (Apply the modifies to the origin)
git pull (Update the local branch from the origin)
git checkout <branchName> (Change branch)

Never use git force push
